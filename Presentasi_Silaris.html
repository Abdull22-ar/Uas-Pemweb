Apa itu Silaris?

Silaris adalah aplikasi web yang memungkinkan masyarakat melaporkan permasalahan sampah secara online.

  • Masyarakat dapat mengirimkan laporan lengkap dengan foto, lokasi/koordinat GPS, dan kategori sampah.
  • Status penanganan dapat dipantau secara real-time menggunakan kode laporan unik (SPH-XXXXXX).
  • Admin/petugas memiliki dashboard untuk mengelola dan merespon laporan.

Frontend dan Backend dibuat terpisah (Decoupled Architecture)
